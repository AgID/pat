# Release PerpetualBeta - 25/06/2020  #
#######################################

Nuove funzioni:
 - [Pre-release] - Nuovo layer DB compatibile PHP 7.3 

Modifiche:
 - Implementato nuovo sistema di crittografia password. Gli algoritmi utilizzati per la crittografia della password sono rispettivamente:
 	  - Bcrypt per la generazione della password concatenata con altri dati;
 	  - Openssl con metodo AES-256-CBC ed hash sha256 per la chiavi univoca del singolo utente.
 	        Funzionamento:
            L'algoritmo di generazione della password crittografata avviene nel seguento modo:
            Passo 1: generazione di una chiave univoca per ogni singolo utente;
            Passo 2: concatenamento della password utente più la chiave interna "ISWEB_KEY";
            Passo 3: creazione e criptazione delle chiavi univoca per ogni utente, casella e-mail e password;
            Passo 4: generazione dell'Hash della password con il concatenamento descrritto nel passo 3.
 - Inibizione esecuzione file eseguibili diretti in cartelle esposte
 - Pulizia file non necessari in distribuzione CKEDITOR incluse
